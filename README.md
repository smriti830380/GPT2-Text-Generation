# GPT-2 Text Generation Project

This project uses GPT-2, a transformer-based language model, to generate human-like text based on a given input prompt.

## 📌 Objective
To train and use a GPT-2 model for generating coherent and contextually relevant text.

## ⚙️ Technologies Used
- Python
- Transformers (Hugging Face)
- PyTorch

## 🚀 How it Works
1. Load pre-trained GPT-2 model
2. Provide an input prompt
3. Model generates text based on the prompt

## 📝 Example

**Input:**
Success is

**Output:**
Success is the most important thing. The most important thing is to be able to do it.
